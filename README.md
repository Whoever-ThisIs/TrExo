# Motivation

TrExo is a code that simulates the exoplanet transit and resulting light curve given its values for semi-major axis (a), excentricity of the orbit (e), inclination of the ecliptic plane respective to the plane of the sky (inc), and planet radius (planet_radius).

# Installation
`TrExo` is (will be) registered on `pip` and works in Python >=3.9. In order to install it, verify that you have the latest versions of `matplotlib` and `numpy` installed.
To install `TrExo`, run in the terminal:
```
$ pip install TrExo
```
